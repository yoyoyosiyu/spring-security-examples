# MyAdmin

MyAdmin 是一个集合了Spring MVC、Spring Security、 Hibernate、 Lomback 等技术的JAVA项目。该项目可以运行在支持JAVA Servlet 3.0 标准的容器中运行。
Sevlet 的配置采用了Java类+XML混合的配置。
